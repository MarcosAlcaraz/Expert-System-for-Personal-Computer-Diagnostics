                    Reemplazar Motherboard{
                        Intenta adquirir el mismo modelo de motherboard, ya sea de laptop o de PC o uno con el socket equivalente al de tu CPU, si el CPU no es modular, adquirir una motherboard con conectores de tu motherboard dañado.
                    }
                    Reemplazar PSU{
                        Adquiere un PSU que sea de la misma o mayor potencia a la dañada. Además verifica que tenga los mismos conectores y factor de forma que la dañada.
                    }
                    Reemplazar HDD{
                        Los discos HDD suelen estropearse con el tiempo, adquiere un SSD, ¿Qué Bus tiene disponible tu Ordenador?{
                            SATA{
                                Adquiere un SSD con puerto SATA, no son los más rápidos pero para usuarios promedio satisface todas sus necesidades, son discos rectangulares delgados de 6.8cm x 3.96cm.
                            }
                            MSATA {
                                Adquiere un SSD que se conecte por MSATA. No son tan comunes pero usan un puerto SATA reducido para tarjetas delgadas con un puerto poco más grande que el M.2. 
                            }
                            M.2 SATA {
                                Adquiere un SSD que use el puerto M.2 pero que sea de interfaz Sata. Son tarjetas delgadas, más chicas que las MSATA y tienen tres Patas, Una chica, una grande y una chica en esa jerarquía.
                            }
                            M.2 NVMe{
                                Adquiere un SSD que use el puerto M.2, son tarjetas delgadas un poco más pequeñas que el MSATA con dos patas, una chica y una grande, así sabemos que es M.2 NVMe y no M.2 SATA.
                            }

                Actualizar Sistema Operativo (SO){
                    ¿Qué sistema operativo usas?{
                        Windows{
                            Busca en la barra de tareas "Buscar Actualizaciones" y presiona "Buscar Actualizaciones". Asegurate de que el sistema no te pida más actualizacioes.
                        }
                        Linux{
                            Independiente de la distribución que se use, accede a la terminal y coloca el comando "sudo apt-get dist-upgrade" para actualizar el sistema operativo. Asegurate de que el sistema no te pida más actualizacioes.
                        }
                        Android{
                            Accede a configuración, busca "Actualizacion del sistema" y toca el botón "Buscar actualizaciones". Asegurate de que el sistema no te pida más actualizaciones.
                        }
                        Mac OS{
                            Busca en Spotlight "Actualización de Software" e inicia el proceso de búsqueda de software nuevo automáticamente. Antes de instalar las actualizaciones te pedirá la contraseña de administrador, la colocas, confirmas y comenzará la instalación de las actualizaciones.
                        }
                        Otro{
                            En promedio, debes acceder a un panel de Actualizaciones y buscas las actualizaciones más recientes automáticamente y se instalan por su cuenta.

        ¿Se apaga?{
            La temperatura es demasiado alta y los ventiladores son muy ruidosos{
                    Limpieza de Ventilador/es y discipador/es{
                        Usa una lata de aire comprimido o una pistola de aire (no secadora de pelo) y sopla en el discipador y el ventilador de tu computadora. Procura que la suciedad salga y no entre al ventilador.
                    }
                    Cambio de pasta térmica{
                        Abre tu computadora, usa las herramientas adecuadas comodestornilladores, algodón, alcohol y pasta térmica de calidad. Retira el sistema de refrigeración de tu CPU y/o GPU, limpia la pasta térmica antigua cuidadosamente y coloca la cantidad de un grano de arroz en el núcleo del CPU o GPU. Coloca y atornilla el discipador de forma uniforme atornillando levemente todos los tornillos hasta que todos queden apretados uniformemente, cierra tu computadora y monitorea las temperaturas con programas como HWInfo.
                    }
                    Revisión de sistema de refrigeración{
                        Revisa que el discipador este bien atornillado, que la pasta térmica esté en buen estado y que no estén golpeados lis pipe lines de cobre.
                    }

        Códigos de error con Sonidos{
            1 largo, dos cortos{
                Fallo en la motherboard o GPU{
                    Verificar las conexiones de GPU y motherboard.
                    Reemplazar la motherboard o GPU si están dañadas.
                }
            }
            3 largos, dos cortos{
                RAM mal colocada o con puertos sucios{
                    Intenta recolocar la Memoria RAM correctamente o limpia los puertos con un paño seco y poco abrasivo.
                }
            }
            4 pitidos{
                Fallo en Chip de Seguridad{
                    Revisar el Chip de seguridad o contactar con soporte técnico de la marca de tu motherboard o Laptop.
                }
            }
        }

    Códigos de error{
        Códigos de Luces{
            2 rojos, 2 blancos{
                Bios corrupto{
                    Recuperar BIOS desde una unidad USB{
                        Accede al soporte del fabricante de tu laptop o motherboard y busca herramientas de recuperación de BIOS.
                    }
                    Restablecer el BIOS con shortcuts{
                        Usa teclas especificas definidas por tu fabricante de motherboard o laptop para presionar durante el arranque para el restablecimiento del BIOS. Usualmente es la tecla "Windows" + "B".
                    }
                }
            }
            3 rojos, dos blancos{
                Fallo de Memoria RAM{
                    Recolocar RAM{
                        Abre tu computadora, quita la RAM y colocala de nuevo, si es posible limpiala cuidadosamente con un paño poco abrasivo.
                    }
                    Cambiar RAM{
                        En caso de que hayas recolocado la RAM y sigue sin funcionar, es probable que una de ellas o la única esté dañada, verifica cuál es la dañada probando una por una hasta que la computadora falle. Además verifica que el Slot no esté sucio ni estpe dañado. Para saber si esá dañado, coloca una RAM que sepas que está buena en ese Slot.
                    }
                }
            }
            3 rojos, 3 blancos{
                Fallo en la GPU{
                    Revisar físicamente la GPU{
                        Abre tu computadora y revisa tu GPU si tiene componentes quemados o dañados. En laptops debes revisar debajo del discipador quitandolo, usualmente está al lado del CPU y debajo del discipador. Si está dañada debes cambiar de GPU o de laptop.
                    }
                    Actualizar Drivers{
                        En caso de que tu Computadora encienda, prueba descargando los drivers correctos y oficiales de tu GPU.
                    }
                    Reempalzar GPU{
                        En caso de que ninguna solución te haya funcionado. Prueba cambiar de GPU por una nueva. O bien, usa los gráficos integrados de tu procesador (APU).
                    }
                }
            }
            5 rojos, 2 blancos{
                Problemas con el controlador integrado{
                    Intenta reiniciar el controlador integrado siguiendo los pasos de recuperación de la página de soporte de tu fabricante de laptop o motherboard.
                }
            }
        }
        Errores POST{
            Error de Suma CMOS{
                Rempalzar batería de CMOS. Usualmentes es una pila de reloj puesta encima de la motherboard. Algunas laptops usan baterías especificas, intenta comprar la oficial para tu laptop. Si tu laptop no tiene batería CMOS, trata de quitar la batería principal de la laptop.
            }
            Sin POST{
                Probar RAM{
                    Recolocar RAM{
                        Abre tu computadora, quita la RAM y colocala de nuevo, si es posible limpiala cuidadosamente con un paño poco abrasivo.
                    }
                    Cambiar RAM{
                        En caso de que hayas recolocado la RAM y sigue sin funcionar, es probable que una de ellas o la única esté dañada, verifica cuál es la dañada probando una por una hasta que la computadora falle. Además verifica que el Slot no esté sucio ni estpe dañado. Para saber si esá dañado, coloca una RAM que sepas que está buena en ese Slot.
                    }
                }
                Reempalzar Motherboard{
                    Contactar con el soporte de tu laptop o motherboard para la revisión de tu motherboard, el cambio de ella o directamente la compra de una nueva laptop. En PC solo es el cambio de motherboard.
                }
            }
